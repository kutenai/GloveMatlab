roty

PURPOSE

ROTY Rotation about Y axis

SYNOPSIS

function r = roty(t)

DESCRIPTION 

ROTY Rotation about Y axis

    R = ROTY(theta)

 Returns a 3x3 rotation matrix representing a rotation of theta 
 about the Y axis.

 See also: ROTX, ROTZ, ROTVEC.
