trotvec

PURPOSE

TROTVEC Rotation about arbitrary axis

SYNOPSIS

function T = trotvec(v, t)

DESCRIPTION 

TROTVEC Rotation about arbitrary axis

     TR = TROTVEC(V, THETA)

 Returns a homogeneous transformation representing a rotation of THETA 
 about the vector V.

 See also: TROTX, ROTX, ROTY, ROTZ.
